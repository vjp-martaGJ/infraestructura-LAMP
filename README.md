# Proyecto: Documentación del Despliegue de Infraestructura LAMP con Monitorización

Este repositorio contiene la planificación, diseño y documentación técnica detallada para la puesta en marcha de la presencia web y el sistema de gestión interna de una PYME.

## 👥 Miembros del Equipo

* **Miembro B:** [Assia Chaibi] - Consultor de Sistemas

## 🏢 Escenario Técnico
La solución propuesta para el cliente consta de:
* **Servidor Web:** Apache con soporte PHP en Ubuntu Server 22.04 LTS.
* **Base de Datos:** MySQL/MariaDB con entornos segregados (Web y Gestión).
* **Seguridad:** Acceso remoto seguro mediante SSH y Firewall básico con UFW.
* **Operaciones:** Monitorización del sistema y estrategia automatizada de backups.
